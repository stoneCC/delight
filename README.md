# delight   一个前端组件库

从事前端这么多年了，一直有个愿望，就是做一个自己的组件库。
接下来，我想去完成自己这个愿望。

> 要达成的目标：
* 同时支持 vue 和 react
* mobile first
* 响应式
* css 系统
* * reset
* * 风格可定义
* * 包含，字体，布局，等
* 配套的css编译处理逻辑
* * webpack
* * sass
* 组件
* * navgator
* * button
* * input
* * radio
* * select
* * textArea
* * form
* * 慢慢增加
* 配套网站
* * 可视化
* * 文档化
* * 代码可拷贝

> 总体思路
* css 参考（抄袭）bootstrap4，但要一句一句的抄
* 采用 ES6
* 采用 webpack4
* 浏览器支持最新的就好
